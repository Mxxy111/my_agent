# Specialist Routing

Use this map before loading other skills. `course-paper-workflow` coordinates the assignment; specialist skills handle focused work.

## Writing And Argument

| Need | Prefer |
|---|---|
| General academic writing | `research-writing`, `scientific-writing` |
| Argument rebuilding or section architecture | `nature-writing` |
| Publication-like English polish | `nature-polishing` |
| Chinese academic polishing | `humanizer-zh`, `nature-polishing` when Chinese-to-English is involved |
| English social-science humanization | `humanize-academic-writing` |
| General AI-flavor diagnosis | `anti-ai-flavor` |

## Sources And Citations

| Need | Prefer |
|---|---|
| Multi-source academic search | `nature-academic-search`, academic research tools, web with official/primary sources |
| Strict claim-to-citation support | `nature-citation` |
| PDF or paper reading | `nature-reader`, `paper-analyzer`, PDF/document skills |
| Zotero or reference manager work | `zotero:Zotero` when available and user approves writes |

## Documents And Deliverables

| Need | Prefer |
|---|---|
| DOCX/Word formatting | `documents:documents` or local document tools |
| PDF final check | PDF tools |
| Slides from paper/report | `nature-paper2ppt`, `paper-deck`, or presentation tools |

## Escalation Rules

Switch to `paper-project-workflow` when the assignment becomes:

- thesis-scale or publication-grade
- data/code-backed original research
- bioinformatics, clinical, medical, or systematic review with formal protocol needs
- reviewer-response, journal submission, or final publication package

Do not load bioinformatics, code-review, data-analytics, or runtime-heavy skills for ordinary course papers unless the user explicitly asks for data/code analysis.
