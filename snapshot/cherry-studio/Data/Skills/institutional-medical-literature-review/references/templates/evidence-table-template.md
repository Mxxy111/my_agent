# Evidence Table Template

Use this template for medical literature reviews after metadata and full text are available.

| Field | Description |
|---|---|
| Citation | First author, year, title shorthand |
| PMID | PubMed identifier |
| DOI | Digital object identifier |
| Source | PubMed, WoS, CrossRef, Zotero, etc. |
| Study design | Trial, cohort, case-control, registry, meta-analysis, methods paper, guideline |
| Population | Disease, inclusion criteria, country/setting |
| Sample size | Number of participants or records |
| Exposure / intervention | Main exposure, treatment, biomarker, diagnostic test, or method |
| Comparator | Comparator group or reference |
| Outcome | Primary and relevant secondary outcomes |
| Main result | Key quantitative or qualitative finding |
| Adjustment variables | Covariates or model details |
| Strengths | Main methodological strengths |
| Limitations | Main caveats, bias risks, missingness, generalizability |
| Relevance | How it supports or challenges the user's project |
| Access status | PMC, OA, institution required, Zotero full text, metadata only |
| Confidence | High, moderate, low; explain briefly |

## Extraction Rules

- Prefer full text over abstract when available.
- If only abstract is available, label the extraction as abstract-based.
- Preserve exact effect estimates and confidence intervals.
- Do not infer unreported adjustment variables or outcomes.
- Keep unsupported claims out of the synthesis.
